Accuracy@K：Top-K 是否完整命中标注相关文档，范围 0-1，最大 1.0，越大越好。
Precision@K：Top-K 中相关文档占比，范围 0-1，最大 1.0，越大越好。
Recall@K：标注相关文档被召回的比例，范围 0-1，最大 1.0，越大越好。
MRR / NDCG@K：相关结果是否排在前面，范围 0-1，最大 1.0，越大越好。
P50 / P95 延迟：典型和长尾耗时，最小 0ms，无理论上限，越小越好。
首字延迟 TTFT：流式首个 token 等待时间，最小 0ms，无理论上限，当前预留，越小越好。
